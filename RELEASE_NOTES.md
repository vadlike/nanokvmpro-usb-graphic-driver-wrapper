# Release Notes

Modified Windows driver package for NanoKVMPro USB Graphic based on the official Sipeed release asset:

`https://github.com/sipeed/NanoKVM-Pro/releases/download/v1.0.5/nanokvmpro_usb_graphic_win.zip`

## Included

- fixed `INF` matching for `USB\VID_3346&PID_1009`
- rebuilt and re-signed driver package for local installation
- added `driver-tool.cmd` with install/remove actions

## Notes

- this is not the untouched official package
- this package uses a local test certificate
- intended for manual Windows installation
